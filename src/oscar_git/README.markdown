#Título:
###Par o impar

___

##Descripción del proyecto:
Este proyecto consiste en un sencillo programa  
que te ayuda a saber si un número de tres  
cifras que le introduzcas es par o impar.

Por pantalla mostrará ***"Es par"*** o ***"NO es par"***.

___

##Cómo instalar:
####Pasos:
1. Instalar java
2. Ejecutar el CMD
3. Compilar el fichero .java con javac
4. Ejecutar con java el fichero creado .class
